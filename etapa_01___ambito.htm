Visão geral desta etapa

 

Para se fechar o âmbito dos trabalhos, fundamental diferenciar 2 conceitos:

 

Product scope

1.O âmbito do Produto, são as funcionalidades e requisitos a serem introduzidos / desenvolvidos nos Produtos da UN Transportes como resultado da Release ou Projeto de Desenvolvimento em questão.
2.O sucesso do desempenho é medido pelo grau de entrega dos requisitos do Produto.

 

Release / Project Scope

1.  O trabalho que precisa de ser feito para se entregar o Produto

2.  O sucesso do desempenho é medido pelo plano da release / projeto em termos de cumprimento do prazo, custo, qualidade e âmbito.
